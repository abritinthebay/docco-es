Defaults

These are the default configuration options. All of these may be extended by user-specified options - either in the CLI or programatically.

export default {
	css: null,
	extension: null,
	languages: {},
	layout: "parallel",
	marked: null,
	output: "docs",
	template: null,
	throw: false,
	verbose: false
};
